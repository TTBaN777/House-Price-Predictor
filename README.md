# House Price Predictor

A Python project for scraping and predicting house prices.

## Project Structure

```
house-price-predictor/
├── data/               # Directory for raw data or database files
├── src/                # Source code directory
│   ├── __init__.py     # Package initialization
│   ├── scraper.py      # Scraper class for web scraping logic
│   ├── database.py     # Database class for data operations
│   └── utils.py        # Common utility functions
├── main.py             # Main entry point for the application
├── requirements.txt    # List of required packages
└── .gitignore          # Git ignore file
```

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/your-username/house-price-predictor.git
   cd house-price-predictor
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

Run the main script:
```
python main.py
```

## Features

- Web scraping for house price data
- Database storage using SQLite
- Utility functions for data processing
